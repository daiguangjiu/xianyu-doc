🔑 第二章 账号注册与基础搭建

2.1 注册流程（标准操作）

  1. 下载闲鱼APP：通过官方应用商店下载，勿用第三方修改版。
  2. 注册/登录：使用手机号注册淘宝账号 → 自动关联支付宝。已有淘宝/支付宝账号可直接登录。
  3. 绑定支付宝：进入「我的」→「设置」→「支付设置」→ 绑定已实名认证的支付宝账号。
  4. 设置头像和昵称：头像用真实/半真实照片（增加信任感），昵称简洁易记，建议与经营类目相关。
  5. 完善个人资料：填写个人简介、所在城市、性别等基础信息。简介中可自然嵌入经营方向。
一个身份证可实名认证 3个支付宝账号，对应可开 3个闲鱼账号。但需配合3个不同手机号，且遵守"一机一卡一号"原则。

2.2 实名认证

认证类型要求路径
基础实名身份证 + 人脸识别我的 → 设置 → 账号与安全 → 实名认证
支付宝实名需与闲鱼账号绑定的支付宝已完成实名支付宝APP → 我的 → 设置 → 账号与安全
企业蓝V认证营业执照 + 法人身份信息需通过代理商/官方渠道提交
建议优先完成支付宝高级实名认证（绑定银行卡），可提升账号权重和交易额度。

2.3 芝麻信用分与闲气值

指标满分合格线影响
芝麻信用分950≥ 650（优秀≥700）影响商品排名权重、买家信任度、超级擦亮投放资格
闲气值100≥ 80（鱼小铺门槛）低于80无法开通鱼小铺；低于60限制发布商品
提升芝麻信用分：完善学历/职业信息、按时还花呗/借呗、多使用支付宝线下支付、保持稳定收货地址。

🏪 第三章 鱼小铺开通全流程

3.1 什么是鱼小铺？

鱼小铺是闲鱼官方推出的专业卖家工具，开通后可获得：

  • 专属店铺页面（自定义封面、分类展示）
  • 商品上限提升至 500个（普通用户仅50个）
  • 数据看板（曝光量、浏览量、转化率等）
  • 付费推广权限（超级擦亮）
  • 优先参与平台活动

3.2 快速开通实操方法（10-16天）

以下为经过验证的加速方法，核心逻辑是向平台证明你是"活跃的专业卖家"。

第一阶段：基础活跃（第1-3天）

  1. 每日签到领闲鱼币，完成「我的」页面全部日常任务（3-5个即可）。
  2. 每日保持 2小时以上在线，浏览推荐页、点赞、关注其他卖家。
  3. 在「会玩」社区发布2-3条帖子，内容与你的经营类目相关。
  4. 完善个人资料：头像、简介、背景图全部设置到位。

第二阶段：买卖养号（第4-10天）

  1. 每天上架10个商品，间隔20分钟发布一个，分散到全天。
  2. 每天完成5笔购买：在闲鱼购买虚拟商品（PPT模板、教程笔记等，单价0.1-1元），买不同店铺的商品。
  3. 每笔确认收货后写10字以上好评，并关注卖家后再评价。
  4. 每天卖出商品：上架虚拟商品（如教程资料），定价0.1元包邮，用闲鱼币推广+加入公益捐赠。有人拍下后发网盘链接，选「无需寄件」发货。
  5. 下架曝光差的商品，优化后重新上架。

第三阶段：冲刺触发（第10-16天）

  1. 保持每日上架+买卖节奏不中断。
  2. 积极回复买家咨询（10分钟内回复加分）。
  3. 主动确认收货 + 给买家好评。
  4. 累计发布商品达到50个以上后，继续发布以刺激系统。
不要大量上架一模一样（同标题、同图片、同文案）的商品，会被判定为重复铺货违规。图片和文案必须做差异化调整。

3.4 常见问题

问题解答
多久能开通？正常2-16天收到邀请，超过30天建议换号重开。
有违规记录能开通吗？1次轻微违规+闲气值≥98仍有概率开通；3次以上违规几乎不可能。

闲鱼搜索算法基于标题关键词匹配。标题 = 流量入口，写好标题等于免费的搜索排名。
要素说明示例

修饰词属性/状态描述全新、九成新、包邮、清仓

9.1 风控预防体系

风控不是事后补救，而是贯穿账号全生命周期的管理。核心原则：环境隔离、行为真实、节奏自然、证据完整。

层级防控点执行标准

行为层自然养号每日在线≥2小时；模拟真实用户浏览、点赞、评论
交易层真实交易避免同一账号高频买卖给相同对象；拒绝刷单、虚假发货
内容层合规发布禁售清单对照；不使用违禁词；不发布站外联系方式
证据层留痕备查发货视频、聊天记录、物流单号保存至少3个月

9.3 风险出现后排查与处置

当收到风控提示后，按以下流程处理，避免情绪化操作导致二次违规：

第一步：确认风险类型

提示文案对应风险类型可逆性
"需完成实名认证解除风险"不当注册可逆（完成认证后可能恢复）
"账号被永久限制使用，请使用其他手机号重新注册"不当使用账户不可逆
"存在危及交易安全"危及交易安全视情节，可申诉
商品被下架 + 扣分发布违禁/风险信息可申诉

第二步：定位触发原因

  1. 检查近7天操作记录：是否批量注册、多账号共用网络、同一设备切换账号、异常IP登录。
  2. 检查交易行为：是否高频买卖给同一对象、是否有买家投诉、是否有虚假发货。
  3. 检查商品内容：是否误触禁售品类、是否使用违禁词、是否引导站外。
  4. 检查矩阵关联：同一身份证/手机号/设备/网络下其他账号是否同时异常。
  5. 检查登录环境：是否使用了VPN/代理IP、是否异地登录频繁。

风险类型处置动作

13.1 卖方软件服务费收费规则

这是所有闲鱼卖家必须了解的核心规则，直接影响利润核算。
费用类型收费门槛费率计算基数
基础软件服务费所有成交订单（无门槛）0.6%每笔订单成交额
软件服务费-基础门槛当月累计成交＞10单 且 累计金额＞10,000元1%（超出门槛部分）超出门槛的每笔订单成交额
软件服务费-品类门槛（手机）当年手机品类累计成交＞6笔1%（超出门槛部分）超出门槛的手机品类每笔订单成交额
软件服务费-鱼小铺开通鱼小铺工具服务的卖方1%（所有订单）每笔订单成交额

关键要点：

  • 总费率上限：同一笔订单最多收取 0.6%（基础）+ 1%（软件服务费）= 合计 1.6%
  • 基础门槛 vs 品类门槛：同一订单同时满足时，仅按品类门槛收取（不重复收）
  • 鱼小铺卖家：不再享受"基础门槛"和"品类门槛"的优惠政策，每笔订单都收取1%+0.6%
  • 退款不退费：成交后发生退款，已划扣的软件服务费不做退还
  • 扣款方式：由支付服务商从买家支付的订单款项中实时划扣
  • 手机品类统计：从每年1月1日起统计交易成功订单，超出6笔后开始收费
  • 规则生效：2024年9月1日起实施基础软件服务费；2026年4月18日最新修订
运营启示：对于月销超1万或手机品类活跃卖家，需将1.6%的服务费纳入定价成本核算。鱼小铺卖家需特别注意——一旦开通，每单都收费，小额多单模式成本更高。
